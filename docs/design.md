# Ancient World Strategy Game Design

## Core Vision
A grand strategy game starting in 0 AD with the entire world map.

## Minimum Viable Product (MVP) Goal
- Control 10 provinces
- Basic economy (food, gold)
- Simple military (levy troops)
- Turn-based (1 turn = 1 month)

## First Prototype Target
A single playable turn with:
1. Map with 3 connected provinces
2. Resource display
3. Basic movement between provinces
